# homebrew-doable

Homebrew tap for the [Doable](https://doable.do) CLI.

## Install

```bash
brew tap doublewltd/doable
brew install doable
```

Once tapped, `doable` is available directly:

```bash
doable login
doable deploy
```

## Updating

```bash
brew update
brew upgrade doable
```

## Other install paths

- npm: `npm install -g doable-cli`
- Source: <https://github.com/doublewltd/doable.do>

## Releases

This tap is updated automatically whenever a new `doable-cli` version is
published to npm — see the `release` workflow in the main repo.
